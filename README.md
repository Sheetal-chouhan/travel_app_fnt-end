A modern **hotel booking web application** built with **React.js**, featuring hotel search, filters, wishlist, booking flow, and responsive UI.  

---

## ✨ Features  

- 🔐 **Authentication** – Signup & Login with form validation  
- 🏨 **Hotel Listings** – Browse & filter hotels  
- 🎚️ **Advanced Filters** – Price, rating, property type, rooms, free cancellation  
- 📅 **Date Selector** – Select check-in & check-out dates  
- 💸 **Final Price Calculation** – View total cost before booking  
- ❤️ **Wishlist** – Save favorite hotels  
- 📜 **Order Summary** – Review booking before payment  
- 💳 **Payment Page** – Mock payment integration  
- 📱 **Responsive UI** – Works across all devices  

---

## 🛠️ Tech Stack  

- **Frontend:** React.js, Context API, Reducer Pattern, CSS  
- **State Management:** Context + Reducers  
- **Deployment:** Render  

---

## 📂 Project Structure  

├── public # Static assets

├── src

│ ├── components # UI components (Navbar, HotelCard, Filters, etc.)

│ ├── context # Global state (Auth, Wishlist, Alerts, Date, Hotel) 

│ ├── pages # App pages (Home, SearchResults, Wishlist, Payment) 

│ ├── reducer # Reducers for state handling

│ ├── services # API calls (login, signup)

│ ├── utils # Regex, helpers, utilities

│ ├── App.js # Root application

│ └── index.js # Entry point
